# 🎮 JOKENPÔ - Pedra, Papel e Tesoura

![.NET](https://img.shields.io/badge/.NET-Console-512BD4?style=flat&logo=dotnet&logoColor=white)

## 📖 Sobre

Jogo clássico **Pedra, Papel, Tesoura** feito em **C# Console**.  
Jogador vs PC com jogadas aleatórias e loop infinito (S/N).

## ✨ Funcionalidades

- ✅ Escolha do jogador
- ✅ PC aleatório (`Random`)
- ✅ Avaliação automática de vencedor
- ✅ Continuar jogo (S/N)
- ✅ Código limpo e modular
